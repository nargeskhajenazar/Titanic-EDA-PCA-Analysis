## هدف پروژه

این پروژه به بررسی و تحلیل اکتشافی دیتاست تایتانیک (Titanic) می‌پردازد. هدف اصلی، درک ساختار داده‌ها، شناسایی الگوهای پنهان بین متغیرها و آماده‌سازی داده‌ها برای مدل‌سازی نهایی با استفاده از تکنیک‌های کاهش ابعاد است. مراحل انجام شده شامل موارد زیر است:

1. **بارگذاری داده:** خواندن فایل CSV دیتاست و بررسی اولیه.
2. **تحلیل اکتشافی (EDA):**
   - بررسی ابعاد و نوع داده‌ها.
   - محاسبه ماتریس همبستگی (Correlation Matrix) برای شناسایی روابط خطی.
   - رسم نمودارهای جعبه‌ای (Boxplot) و هیستوگرام برای بررسی توزیع و شناسایی Outliers.
   - رسم نمودارهای رگرسیون برای بررسی رابطه متغیرها با متغیر هدف (Survived).
3. **کاهش ابعاد (Dimensionality Reduction):**
   - اجرای الگوریتم PCA (Principal Component Analysis) برای کاهش ابعاد داده‌ها به ۲ بعد جهت مصورسازی بهتر.
   - استانداردسازی داده‌ها با استفاده از StandardScaler.

## کتابخانه‌ها

- **Pandas**: برای مدیریت و پردازش داده‌ها
- **Seaborn & Matplotlib**: برای مصورسازی داده‌ها (Heatmap, Boxplot, Histogram, Regplot)
- **Scikit-learn**: برای استانداردسازی داده‌ها و اجرای الگوریتم PCA
